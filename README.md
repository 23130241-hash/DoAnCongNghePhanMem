# DoAnCongNghePhanMem
Game thủ thành (Tower Defense)
1. Format commit
<type>(<scope>): <short description>
type:
feat – thêm tính năng gameplay
fix – sửa lỗi game, bug AI, pathfinding, hoặc UI
refactor – tối ưu code game, tái cấu trúc script, không thay đổi gameplay
style – chỉnh UI, hiệu ứng, animation, màu sắc
docs – cập nhật tài liệu, hướng dẫn chơi, README
test – thêm/sửa test cho gameplay, tower, enemy, wave
chore – thay đổi config game, assets, build scripts
scope: module hoặc feature cụ thể, ví dụ: tower, enemy, wave, ui, level
short description: câu ngắn, dùng động từ hiện tại, cụ thể, không chung chung
  thống nhất viết bằng tiếng việt
